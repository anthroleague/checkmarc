Download Download View Checkmarc on GitHub

What

A way to build reports that check your batch MARC records for common problems.

How

Start a report, add some checks, upload a batch marc file, get back your report on what needs human intervention.

Why

Do you get dirty MARC data from any of your data providers? Need to check it before it goes public? Run a report to find out what records and fields might actualy need intervention.
